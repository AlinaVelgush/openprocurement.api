Qualification
=============

Schema
------

:id:
    string, autogenerated


:bidID:
    string, autogenerated

:date:
    string, :ref:`Date`, autogenerated, read-only

    The date of the qualification.

:documents:
    List of :ref:`Document` objects

:status:
    string

    Possible values are:

    * `pending`
    * `active`
    * `unsuccessful`
    * `cancelled`

Workflow
--------

.. graphviz::

    digraph G {
        A [ label="pending*" ]
        B [ label="active"]
        C [ label="cancelled"]
        D [ label="unsuccessful"]
         A -> B;
         A -> C;
         A -> D;
         D -> C;
         B -> C;
    }

\* marks initial state
